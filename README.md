# fishypi
A Rasberry Pi PHP app for remote management of an Aquaponics system.

##Hardware Spec
 - [Raspberry Pi Model A+] (other models may work as well)
 - JBtek 8 Channel DC 5V Relay Module
 
##Software
 - [Raspbian Linux]
